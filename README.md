# Rust Ray Tracer

## Overview
This is a simple ray tracer implemented in rust, following the *Ray Tracing in One Weekend* by Peter Shirley. 
http://in1weekend.blogspot.ca/2016/01/ray-tracing-in-one-weekend.html
